# Justice Driven Task Power

Justice Driven Task Power es una aplicación para organizar y motivar tu día a día de forma sencilla y visual.

## ¿Qué puedes hacer?

- **Tareas**: Crea, edita y marca tareas diarias. Lleva el control de lo que tienes pendiente y lo que ya lograste.
- **Hábitos**: Registra hábitos que quieres mantener. Marca cada vez que los cumples y observa tu progreso a lo largo del tiempo.
- **Metas**: Define metas personales, ya sean para aumentar (como leer más libros) o reducir (como bajar gastos). El sistema te muestra claramente cuándo una meta está en progreso y cuándo la has cumplido.

## ¿Cómo funciona?

- Visualiza todas tus tareas, hábitos y metas en un solo lugar.
- Marca tareas y hábitos conforme los realizas y observa cómo avanzas.
- Las metas pueden ser de dos tipos:
  - **Incrementales**: Cumples la meta cuando alcanzas o superas el objetivo (por ejemplo, ahorrar cierta cantidad).
  - **Reductuales**: Cumples la meta cuando logras reducir el progreso hasta igualar o estar por debajo del objetivo (por ejemplo, bajar de peso).
- El sistema te motiva mostrando tu progreso y celebrando cada meta cumplida.

¡Organiza tu vida y alcanza tus objetivos de manera clara y motivadora!
